# CSS Tips

En statisk webbplats med korta CSS-tips inspirerade av
[j4rl/css-tips](https://github.com/j4rl/css-tips).

## Innehåll

- Sökbar tipskatalog
- Filter för kategori och nivå
- Dagens CSS-tips
- Progression med två tips per vecka
- Support/fallback, challenge och läraranvändning per tips
- Kopiera CSS, kopiera lektionsunderlag och öppna exempel i CodePen
- Undersidor för katalog, progression, labb och varje enskilt tips
- Tema: auto, ljust och mörkt, där auto följer systeminställningen
- `contrast-color()` som progressiv förbättring för knappar och badges

Öppna `index.html` direkt, eller gå till `http://localhost/css-tip/` om projektet ligger under XAMPP och Apache är igång.
